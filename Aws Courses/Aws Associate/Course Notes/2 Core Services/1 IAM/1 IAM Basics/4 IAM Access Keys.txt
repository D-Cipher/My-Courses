IAM Access Keys

IAM Access keys -> :ong-term credentials for an IAM user for programmatic requests to the AWS CLI or AWS API (directly or using the AWS SDK). 
- Users need their own access keys to make programmatic calls to AWS
- Secret access kyes should be saved in a secure location

Calls can come from:
- AWS Command Line Interface (AWS CLI)
- AWS SDKs
- Windows PowerShell
- Direct HTTP calls using the APIs for individual AWS services
